# Amazon Listing Best Practices

## Title Best Practices

### Formula
```
[Brand] + [Product Type] + [Top Feature] + [Material/Variant] + [Size] + [Color/Quantity]
```

### Do's
- Start with brand name
- Include 2-3 primary keywords
- Use title case (not ALL CAPS)
- Include size, color, quantity
- Use dashes or pipes as separators

### Don'ts
- Exceed 200 characters
- Use promotional language
- Include price/shipping info
- Use special characters excessively

---

## Bullet Points Best Practices

### Formula
```
[CAPS FEATURE] - [Benefit statement] [keyword integration]
```

### Recommended Structure
1. **Bullet 1:** Main benefit/unique selling point
2. **Bullet 2:** Quality/materials
3. **Bullet 3:** Size/dimensions/specs
4. **Bullet 4:** Use cases/versatility
5. **Bullet 5:** Warranty/guarantee/trust

### Do's
- Use CAPS for feature headers
- Focus on benefits, not just features
- Include 2-3 keywords per bullet
- Keep each bullet 200-500 characters

### Don'ts
- Use emojis (can cause issues)
- Start every bullet the same way
- Be too technical without benefit
- Leave bullets unused

---

## Description Best Practices

### A+ Content (Brand Registered)
- Use comparison charts
- Include lifestyle images
- Tell brand story
- Show product in use

### Plain Text Description
- Use short paragraphs
- Include long-tail keywords
- Add call-to-action
- Supplement (don't repeat) bullets

---

## Image Best Practices

### Recommended Count: 7-9 images

### Image Types
1. **Main image:** White background, product only
2. **Lifestyle:** Product in use
3. **Infographic:** Specs, features callouts
4. **Size reference:** With common objects
5. **Close-ups:** Detail shots
6. **Packaging:** What customer receives
7. **Video thumbnail:** If video available

### Requirements
- 1000x1000 pixels minimum (1500x1500 recommended)
- White background for main image
- No text/logos on main image
- JPEG or PNG format

---

## Scoring Guide

| Component | Excellent (80+) | Good (60-79) | Needs Work (<60) |
|-----------|-----------------|--------------|------------------|
| Title | 150-200 chars, structured | 100-150 chars | <100 chars |
| Bullets | 5 bullets, CAPS headers | 5 bullets, basic | <5 bullets |
| Description | A+ Content | 1000+ chars | None or minimal |
| Images | 7+ with video | 5-6 images | <5 images |
