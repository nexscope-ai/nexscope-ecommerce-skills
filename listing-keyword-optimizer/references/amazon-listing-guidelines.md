# Amazon Listing Guidelines

## Title Requirements

### Character Limits by Category
- Most categories: 200 characters
- Clothing/Shoes: 80 characters
- Mobile display: ~80 characters (truncated)

### Title Format
```
[Brand] + [Product Type] + [Key Feature] + [Material/Variant] + [Size/Color] + [Quantity]
```

### Do's
- Put most important keywords first
- Use proper capitalization (Title Case)
- Include key product attributes
- Use hyphens to separate concepts

### Don'ts
- ALL CAPS
- Special characters (★, ✓, ©)
- Promotional phrases ("Sale", "Free Shipping")
- Subjective claims ("Best", "Top-Rated")
- HTML or encoded characters

---

## Bullet Points

### Limits
- 5 bullets standard
- 500 characters per bullet (1000 for some categories)
- Mobile shows first 3 bullets

### Format
```
[FEATURE IN CAPS] - Benefit description with keywords naturally included
```

### Best Practices
- One key feature per bullet
- Start with benefit, not feature
- Include 2-3 keywords per bullet
- Use consistent format

---

## Product Description

### Limits
- 2000 characters (text only)
- A+ Content replaces description (not indexed!)
- EBC/A+ up to 5 modules

### Best Practices
- Use natural sentences
- Tell a story
- Include long-tail keywords
- Avoid duplicate title keywords

---

## Backend Search Terms

### Limits
- 250 bytes (not characters!)
- Single field (no comma separation needed)
- Spaces separate terms

### Include
- Misspellings
- Synonyms
- Alternate names
- Spanish terms (for US)
- Abbreviations

### Avoid
- Brand names (yours or others)
- ASINs
- "Amazon" or promotional terms
- Repetition of front-end keywords
- Subjective claims

---

## Keyword Indexing Rules

1. **Single word indexing**: "yoga mat" indexes for "yoga" and "mat" separately
2. **No need to repeat**: If in title, no need in bullets
3. **Plurals indexed**: "mat" indexes for "mats"
4. **48-hour indexing**: Changes take time
5. **A+ Content NOT indexed**: Description keywords matter if no A+

## Category-Specific Notes

### Electronics
- Include compatibility info
- Model numbers important
- Specs in bullets

### Clothing
- Size chart keywords
- Color names standardized
- Material first

### Home & Kitchen
- Dimensions crucial
- Use case keywords
- Material/finish

### Sports & Outdoors
- Activity keywords
- Level (beginner/pro)
- Certification mentions
